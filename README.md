1. Hiểu rõ cú pháp và các lệnh cơ bản của Bash

Nắm được cấu trúc cú pháp của shell script (shebang, comment, permission…)

Thực hiện thành thạo các lệnh thường dùng: ls, cd, cp, mv, grep, cat, awk, sed, chmod, chown,…

Hiểu sự khác nhau giữa Bash interactive và Bash script.

Sử dụng được các toán tử số học, chuỗi và logic trong Bash.

2. Viết script để tự động hóa các tác vụ lặp đi lặp lại

Tạo ra các script giúp giảm thiểu thao tác thủ công.

Tự động hóa các công việc thực tế: backup file, kiểm tra log, tạo user, kiểm tra hệ thống, deploy ứng dụng.

Lên lịch chạy script định kỳ bằng cron.

Tối ưu hóa quy trình làm việc với automation.

3. Nắm vững biến, điều kiện, vòng lặp và hàm trong Bash

Biết cách khai báo và sử dụng biến cục bộ, biến môi trường.

Điều khiển luồng chương trình bằng if/else, case.

Sử dụng các vòng lặp for, while, until.

Tạo và sử dụng function để tổ chức mã sạch, dễ tái sử dụng.

Truyền tham số vào script qua $1, $2, $@…

Xử lý input từ user và từ file.

4. Xử lý lỗi và debug script hiệu quả

Kiểm tra và xử lý lỗi với exit codes, set -e, set -u, trap.

Thêm logging, timestamp để theo dõi hoạt động script.

Debug script với bash -x / set -x.

Viết script theo best-practices: cấu trúc rõ ràng, dễ bảo trì, an toàn khi chạy.
